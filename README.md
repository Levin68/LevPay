# LevPay - QRIS Payment System

Modern QRIS payment interface dengan fitur lengkap:
- ✅ Real API Integration  
- ✅ Progressive Loading
- ✅ Bottom Sheet Modal (native feel)
- ✅ Auto-save QR to Gallery
- ✅ Dark/Light Mode
- ✅ Confetti Animation
- ✅ Dashboard Statistics
- ✅ PWA Ready

## Setup

1. Clone/download semua file sesuai struktur
2. Taruh logo wallet di `assets/images/wallet-logo/`
3. Jalankan web server: `python -m http.server 8080`
4. Buka `http://localhost:8080`

## API Mode

- **REAL API**: Set `API_BASE` di `js/config.js`
- **DEMO MODE**: Kosongkan `API_BASE`

## Features

- Real-time payment status polling
- Auto-refresh transaction history  
- Template favorit nominal
- Export & print QR
- WhatsApp integration
- Progressive Web App (PWA)
- No minimum amount limit
